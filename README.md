### MarkDown
点击 `create repository` 后，`github`慷慨的赠送了一个`README.md`。  
这时的我在心中默念：这是什么鬼？我马不停蹄的找到了度娘，在她给我的网页里我熟练的跳过了前三页的广告，终于在第四页找到了答案，这是`MarkDown`语言！它是一种轻量级的标记语言，它允许人们使用易读易写的纯文本格式编写文档。

### 编辑工具
`github`本身也可以编辑，但是并没有什么辅助功能。所以工欲善其事必先利其器，互联网上有些编辑工具能够给你更美好的编码体验，帮助你更得心应手的编写MarkDown代码，总之在互联网时代，这不是什么难事。

* [马克飞象](https://maxiang.io/)
* [盘道](http://pandao.github.io/editor.md/)
